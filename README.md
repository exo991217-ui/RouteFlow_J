# RouteFlow ✈️

여행 일정을 일정표 · 이동경로 · 예산으로 한눈에 정리하는 여행 플래너 앱이에요.  
ChatGPT에서 만든 여행 요약 텍스트를 붙여넣는 것만으로 전체 일정이 자동 완성돼요.

---

## ✈️ 여행 가져오기

ChatGPT 등 AI가 생성한 여행 요약 텍스트를 붙여넣으면 앱이 자동으로 분석해요.

| 분석 항목 | 내용 |
|-----------|------|
| 여행 기간 | 시작일 · 종료일 자동 인식 |
| 여행지 | 국가 이름 + 국기 이모지 자동 매핑 |
| 동반자 | 가족 / 커플 / 친구 등 자동 인식 |
| 일정 항목 | 날짜 · 시간 · 장소 · 메모 · 카테고리 · 교통수단 파싱 |
| 지출 내역 | 날짜 · 카테고리 · 항목명 · 금액 파싱 |

가져오기 전에 **카테고리별 미리보기**가 표시되고, 원하는 항목만 선택해서 가져올 수 있어요.

---

## 📅 일정 관리

날짜별 탭으로 나눠서 일정을 확인하고 수정할 수 있어요.

| 기능 | 설명 |
|------|------|
| 날짜 탭 이동 | 날짜를 탭으로 전환하거나 좌우로 스와이프해서 이동 |
| 일정 추가 | 각 날짜에 새 일정 항목 직접 추가 가능 |
| 항목 수정 | 시간 · 장소 · 메모 · 카테고리 · 교통수단 · Google Maps 링크 편집 |
| 항목 삭제 | 개별 항목 삭제 또는 전체 날짜 일괄 삭제 |
| 카테고리 필터 | 관광 / 식사 / 이동 / 숙소 등 카테고리별 필터링 |
| Google Maps 연결 | 장소에 Maps 링크 저장 시 지도 바로가기 버튼 표시 |

---

## 🗺️ 이동경로 시각화

일정의 이동 항목을 자동으로 분석해서 흐름을 카드로 시각화해요.

| 기능 | 설명 |
|------|------|
| 자동 경로 생성 | 이동 항목의 장소와 메모를 분석해서 출발지 → 도착지 자동 추출 |
| 소요시간 계산 | 다음 일정까지의 시간 차이 또는 교통수단 기본값으로 자동 계산 |
| 날짜별 그룹 | 날짜를 선택해서 해당 날의 이동 흐름만 모아 보기 |
| 수동 경로 추가 | 자동 생성 외에 수동으로 이동 카드 직접 추가 가능 |
| 상세 정보 입력 | 환승 여부 · 환승 위치 · 탑승 위치(alight) · 플랫폼 · 이동 거리(km) 입력 |
| 경로 별표 | 자주 참고할 경로에 별표(⭐) 표시 |

---

## 💰 예산 관리

지출 내역을 항목별로 기록하고 전체 예산 현황을 한눈에 파악해요.

| 기능 | 설명 |
|------|------|
| 지출 항목 추가 | 날짜 · 카테고리 · 항목명 · 금액 입력 |
| 카테고리별 합계 | 교통 / 숙소 / 식비 / 관광 등 카테고리별 금액 및 비율 표시 |
| 카테고리 차트 | 전체 지출 중 각 카테고리가 차지하는 비율을 바 차트로 표시 |
| 일별 지출 추이 | 날짜별 지출 금액 변화를 라인 그래프로 시각화 |
| 총 예산 대비 | 여행 총 예산 대비 현재까지 사용 금액과 남은 금액 표시 |

---

## ☁️ Google 로그인 & 클라우드 동기화

Google 계정으로 로그인하면 여행 데이터가 클라우드에 자동으로 백업돼요.

| 상황 | 동작 |
|------|------|
| 처음 로그인 | 기기에 저장된 기존 여행이 클라우드로 자동 업로드됨 |
| 여행 추가 / 수정 / 삭제 | 변경 사항이 즉시 클라우드에 반영됨 |
| 다른 기기에서 로그인 | 동일 Google 계정으로 로그인 시 저장된 여행 목록이 그대로 표시됨 |
| 로그아웃 상태 | 클라우드 없이 기기 로컬 저장으로만 동작 (기능 제한 없음) |

> 데이터는 Firestore에 저장되며, 본인 계정만 접근할 수 있어요.

## 🔗 공유 & 인쇄

| 기능 | 설명 |
|------|------|
| 텍스트로 복사 | 여행 일정 전체를 텍스트 형식으로 클립보드에 복사 |
| 일정 인쇄 | 전체 여행 일정을 A4 세로로 인쇄 |
| 이동경로 인쇄 | 선택한 날짜의 이동 흐름을 A4 가로로 인쇄 |
| 전체 인쇄 | 일정 + 모든 날짜 이동경로 한 번에 인쇄 |

## 🎨 테마 색상

헤더 우측 색상 버튼을 눌러 앱 전체 색상을 바꿀 수 있어요. 선택한 테마는 자동 저장돼요.

| 테마 | 대표 색상 |
|------|-----------|
| 퍼플 (기본) | 보라 계열 |
| 핑크 | 분홍 계열 |
| 민트 | 초록 계열 |
| 블루 | 파랑 계열 |
| 피치 | 주황 계열 |

## 📱 오프라인 지원

앱을 한 번 열면 브라우저에 캐시되어, 인터넷이 없는 환경에서도 저장된 일정을 확인하고 수정할 수 있어요. 해외 여행 중 와이파이 없이도 동작해요.
